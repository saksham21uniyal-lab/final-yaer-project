RUN SYMPTRACK APP - FIXED COMMANDS (Windows PowerShell/CMD)

**PowerShell (VSCode default): Use ; NOT &&**
cd symptrack_flutter; flutter pub get; flutter analyze; flutter run -d chrome

**CMD (.bat files): Double-click**
.\run_flutter.bat  (runs app)
.\analyze_flutter.bat  (lint check)
.\run_backend.bat  (chat server)

**Backend (chat):**
cd symptrack_flutter\backend; pip install -r requirements.txt; python chat_server.py

**TROUBLESHOOT:**
- && error? Use ; in PowerShell
- Flutter not found? Install: flutter.dev
- Deps? flutter pub get
- Outdated deps OK, ignore.
